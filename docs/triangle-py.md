# Описание triangle.py

## def area(a, h)
Возвращает площадь треугольника.

Параметры:
- a: *длина стороны треугольника (положительное число).*
- h: *длина высоты треугольника, проведенной к стороне с длиной 'a' (положительное число).*

Возвращаемое значение:
- *Площадь треугольника со стороной длины 'a' и высотой длины 'h', проведенной к стороне с длиной 'a'.*

Пример вызова:
```python
x = area(5, 10)
print(x) # Output: 25.0
```


## def perimeter(a, b, c)
Возвращает периметр треугольника.

Параметры:
- a: *длина первой стороны треугольника (положительное число).*
- b: *длина второй стороны треугольника (положительное число).*
- c: *длина третьей стороны треугольника (положительное число).*

Возвращаемое значение:
- *Периметр треугольника с длинами сторон 'a', 'b' и 'c'.*

Пример вызова:
```python
x = perimeter(3, 5, 4)
print(x) # Output: 12
```
