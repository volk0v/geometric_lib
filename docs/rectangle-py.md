# Описание rectangle.py


## def area(a, b)
Возвращает площадь прямоугольника.

Параметры:
- a: *длина стороны прямоугольника (положительное число).*
- b: *длина стороны, перпендикулярной стороне прямоугольника с длиной 'a' (положительное число).*

Возвращаемое значение:
- *Площадь прямоугольника с длинами сторон 'a' и 'b'.*

Пример вызова:
```python
x = area(5, 10)
print(x) # Output: 50
```

## def perimeter(a, b)
Возвращает периметр прямоугольника.

Параметры:
- a: *длина стороны прямоугольника (положительное число).*
- b: *длина стороны, перпендикулярной стороне прямоугольника с длиной 'a' (положительное число).*

Возвращаемое значение:
- *Периметр прямоугольника с длинами сторон 'a' и 'b'.*

Пример вызова:
```python
x = perimeter(5, 10)
print(x) # Output: 30
```
