# Описание square.py

## def area(a)
Возвращает площадь квадрата.

Параметры:
- a: *длина стороны квадрата (положительное число).*

Возвращаемое значение:
- *Площадь квадрата со стороной длины 'a'.*

Пример вызова:
```python
x = area(10)
print(x) # Output: 100
```


## def perimeter(a)
Возвращает периметр квадрата.

Параметры:
- a: *длина стороны квадрата (положительное число).*

Возвращаемое значение:
- *Периметр квадрата со стороной длины 'a'.*

Пример вызова:
```python
x = perimeter(10)
print(x) # Output: 40
```
