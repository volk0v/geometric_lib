# Описание square.py

## def area(a)
Возвращает площадь квадрата.

Параметры:
- a: *сторона квадрата.*

Возвращаемое значение:
- *Площадь квадрата со стороной 'a'.*


## def perimeter(a)
Возвращает периметр квадрата.

Параметры:
- a: *сторона квадрата.*

Возвращаемое значение:
- *Периметр квадрата со стороной 'a'.*
