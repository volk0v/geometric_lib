# Описание square.py

## def area(a)
Возвращает площадь квадрата.

Параметры:
- a: *длина стороны квадрата.*

Возвращаемое значение:
- *Площадь квадрата со стороной длины 'a'.*


## def perimeter(a)
Возвращает периметр квадрата.

Параметры:
- a: *длина стороны квадрата.*

Возвращаемое значение:
- *Периметр квадрата со стороной длины 'a'.*
