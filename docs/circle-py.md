# Описание circle.py


## def area(r)
Возвращает площадь круга.

Параметры:
- r: *радиус круга (положительное число).*

Возвращаемое значение:
- *Площадь круга с радиусом 'r'.*

Пример вызова:
```py
x = area(10)
print(x) # Output: 314.1592653589793
```


## def perimeter(r)
Возвращает периметр круга.

Параметры:
- r: *радиус круга (положительное число).*

Возвращаемое значение:
- *Периметр круга с радиусом 'r'.*

Пример вызова:
```python
x = perimeter(10)
print(x) # Output: 62.83185307179586
```
